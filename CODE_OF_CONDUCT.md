# Code of Conduct

This project adopts the **Contributor Covenant v2.1** in full.

The canonical text is here:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

A plain-language summary:

- Be respectful. Disagree about ideas, not people.
- No personal attacks, harassment, or unwelcome conduct.
- Constructive, professional communication only — in issues,
  PRs, discussions, and any community space tied to this project.
- Maintainers reserve the right to remove comments, commits, code,
  issues, and PRs that don't align with this Code of Conduct, and
  to ban participants whose behavior they deem inappropriate.

## Scope

Applies to all project spaces (GitHub repo, issues, PRs,
discussions) and to any individual representing the project in
public.

## Reporting

If you experience or witness conduct that violates this Code,
report it privately to the maintainer at
**nautiyalanupam98@gmail.com**. All reports will be reviewed
and investigated promptly and fairly. Maintainers are obligated
to respect the privacy and security of the reporter.

## Enforcement

Maintainers follow the
[Contributor Covenant Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines):

1. **Correction** — private written warning.
2. **Warning** — public warning with specified consequences.
3. **Temporary ban** — from project spaces for a defined period.
4. **Permanent ban** — for sustained or severe violations.

The version in force is whatever
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
points to at the time the conduct occurred.
